# Parking Management System

A desktop application built with Java Swing for managing parking zones and tracking active vehicle tickets. 

* **Language:** Java 8
* **UI Framework:** Java Swing
* **Database:** MongoDB
* **Build Tool:** Maven
* **Testing:** JUnit 4, AssertJ Swing, Testcontainers, Mockito
